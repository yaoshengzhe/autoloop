Autoloop

Let your AI agent work autonomously until the job is done.

Walk away from your keyboard. Return to working code.

Quick Start

# Install
claude plugin marketplace add yaoshengzhe/autoloop
claude plugin install autoloop@autoloop

# Run
/autoloop Build a REST API with tests --completion-promise 'DONE' --max-iterations 15

Autonomous

Define your goal and let the agent work independently through iterations.

Self-Correcting

Each iteration builds on the last. The agent sees its work and fixes issues.

Safe

Max iterations prevent runaway loops. You stay in control.

Transparent

All work preserved in files and git history for your review.

Example

/autoloop "Build user authentication:
- Login/logout endpoints
- JWT tokens
- Unit tests with 80% coverage

Run tests after each change.
<promise>AUTH COMPLETE</promise>" --max-iterations 20
